# 개인정보 처리방침

**최종 업데이트:** 2026년 3월

## 1. 소개

SafeScribe는 개인정보 보호를 최우선으로 하는 AI 음성 변환 앱입니다. 본 방침은 수집하는 데이터, 처리 방법 및 개인정보에 대한 귀하의 권리를 설명합니다.

## 2. 수집 데이터

| 범주 | 데이터 | 목적 |
|------|--------|------|
| 인증 | Google/Apple 계정 ID, 이메일 | 계정 식별, API 접근 |
| 오디오 | 업로드된 오디오 파일 | 변환 처리 |
| 변환 결과 | 생성된 텍스트, 세그먼트 | 사용자에게 전달 |
| 사용량 | 시간, 비용, 타임스탬프 | 청구 및 추적 |
| 진단 | 충돌 보고서, 오류 로그 | 앱 안정성 |

**수집하지 않는 항목:** 연락처, 위치, 검색 기록, 광고용 기기 식별자, 생체 데이터.

## 3. 오디오 처리 — 제로 디스크 정책

- 오디오는 서버의 **RAM(임시 메모리)에서만** 처리됩니다.
- 오디오는 처리의 **어떤 단계에서도 디스크에 기록되지 않습니다**.
- 변환이 완료되거나 실패하면 모든 오디오 데이터가 **메모리에서 즉시 삭제**됩니다.
- 오디오 콘텐츠의 복사본, 백업 또는 로그는 생성되지 않습니다.

## 4. 로컬 저장소

모든 로컬 데이터는 **AES-256 암호화** 컨테이너에 저장됩니다. 암호화 키는 플랫폼 네이티브 보안 저장소(Android Keystore / iOS Keychain)에 의해 보호됩니다.

**기기에 저장되는 데이터 (암호화됨):**

| 데이터 | 세부사항 | 보존 기간 |
|--------|---------|----------|
| 변환 결과 | 텍스트 내용, 타임스탬프가 있는 세그먼트, 파일 이름, 길이, 비용, 감지된 언어, 파일 해시 | 삭제할 때까지 또는 설정된 보존 기간 후 자동 정리 |
| 오프라인 대기열 | 오디오 파일 경로, 길이, 언어 | 임시 — 업로드 성공 후 제거 |
| 앱 설정 | 변환 언어 기본 설정, UI 로케일, 온보딩 상태, GDPR/Sentry 동의 및 타임스탬프 | 로그아웃하거나 계정을 삭제할 때까지 |
| 중복 감지 인덱스 | 파일 해시 → 변환 결과 ID 매핑 | 변환 결과와 함께 자동 관리 |
| 미완료 작업 상태 | 진행 중인 변환 복구 데이터 | 작업 완료 또는 취소 시 지워짐 |

**플랫폼 보안 저장소의 데이터 (Keychain / Keystore):**

| 데이터 | 용도 |
|--------|------|
| Hive 암호화 키 | 모든 로컬 데이터베이스 컨테이너 암호화 |
| 인증 토큰 | Google/Apple 로그인의 OIDC 토큰 |

- 로컬에 **사용자 프로필이 저장되지 않습니다** (이름, 사진, 전화번호 또는 주소 없음).
- 오디오 녹음은 업로드 후 기기에서 삭제됩니다.
- 계정 삭제 시 모든 로컬 데이터가 영구적으로 삭제됩니다.

## 5. 인증

- Google Sign-In 및 Sign in with Apple을 통한 OpenID Connect(OIDC)를 사용합니다.
- 인증에 필요한 정보만 수신합니다 (사용자 ID, 이메일, 표시 이름).
- 연락처, 캘린더 또는 기타 계정 데이터에 접근하지 않습니다.

## 6. 결제 데이터

- 인앱 구매는 Apple App Store 또는 Google Play Store에서 전적으로 처리합니다.
- SafeScribe는 신용카드 번호나 결제 수단 세부정보를 **수신, 저장 또는 처리하지 않습니다**.
- 잔액 확인을 위한 구매 영수증만 수신합니다.

## 7. 서버측 처리

- 오디오 파일은 인증서 고정이 적용된 TLS 암호화 연결을 통해 업로드됩니다.
- 오디오는 RAM에서만 처리되며 **디스크에 접촉하지 않습니다**.
- 변환 결과는 다운로드를 위해 일시적으로 보관되며 **확인 후 삭제**됩니다.
- 서버측 처리는 음성 인식에 자체 호스팅 오픈소스 AI 모델을 사용합니다.

## 8. 오류 추적

- 충돌 보고 및 오류 모니터링에 Sentry를 사용합니다.
- 모든 로그는 전송 전에 개인 식별 정보(PII)가 제거됩니다.
- 다음 데이터 유형이 마스킹됩니다: 이메일 주소, 전화번호, IP 주소, 파일 경로, 인증 토큰 및 작업 ID.

## 9. 데이터 공유

마케팅 목적으로 개인 데이터를 제3자에게 **판매, 임대 또는 공유하지 않습니다**.

데이터는 다음과만 공유됩니다:
- **변환 서버** (자체 인프라) — 오디오 처리용
- **Sentry** — 충돌 보고용 (PII 제거됨)
- **Apple/Google** — 인증 및 결제 처리용

## 10. 사용자 권리

다음 권리가 있습니다:
- 로컬에 저장된 변환 결과에 언제든지 **접근**
- 개별 변환 결과 또는 앱의 모든 데이터를 **삭제**
- 공유/복사 기능을 통해 변환 결과를 **내보내기**
- 언제든지 **로그아웃**하고 API 접근 권한을 취소
- **계정 삭제 요청** — 로그인 ID, 모든 로컬 저장 데이터 및 남은 크레딧 잔액이 영구적으로 제거됩니다
